窗口组件说明：
    参数配置+按钮
    进度条
    INFO+ERROR日志框
    WARNING日志框
参数配置说明
    CAR路径：缺陷报告路径，一般结尾是aoicar
    JOB路径：资料路径，一般结尾是aoijob
    保存路径：拷贝资料的保存路径
    开始时间：拷贝资料的开始日期
    结束时间：拷贝资料的结束日期
    存储方式：资料存储方式包括
        1.按日期存储
        2.按料号存储
        3.按批量存储
        4.手动机存储
    拷贝方式：资料拷贝方式包括
        1.按日期拷贝：拷贝开始时间到结束时间的所有资料，每个料号拷贝多少板取决于最大板数的值
        2.按板号拷贝：拷贝开始时间从起始板号到结束板号的资料
        3.定时拷贝：如果开始日期设定为当前日期之前，则先拷贝开始日期到当天的资料，然后将开始日期设定为当前日期，
            反之拷贝当前日期的资料
        4.一键拷贝：按照mes系统提供的拷贝配置进行拷贝，拷贝配置仅能使用一次，使用完之后再想使用，需要使用mes系统重新导出拷贝
    起始板号：按板号拷贝 从该值的板号开始拷贝
    结束板号：按板号拷贝 拷贝到该值的板号结束
    最大板数：按日期拷贝 每个料号拷贝的最大数量即每个料号最多拷多少板(有批量文件夹的每个批量最多拷多少版)
按钮说明
    ...:打开本地资源路径
    开始拷贝：按照拷贝方式进行拷贝
    停止拷贝：停止拷贝和定时任务
